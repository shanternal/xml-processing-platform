# XML Processing Platform

Проект предназначен для конвертации XML в JSON.

- В Сервисе 2 (xml2json-service):
- - Эксгибиционизм в логах. Мы настолько открыты миру, что пишем в логи токены, куки и прочие интимные заголовки без цензуры. У нас даже есть специальный тест authorizationHeaderIsNotMasked, который заботливо охраняет наше конституционное право сливать секреты. Надо бы купить сервису одежду и научить маскировать хотя бы Authorization и Cookie.
- - XML-матрёшка смерти (DoS для самых маленьких). Наша конвертация ловит только JSONException, поэтому если скормить сервису XML-матрешку вида ```<a><a><a>...</a></a></a>``` с тысячами уровней вложенности, парсер выплюнет великий и ужасный StackOverflowError. А поскольку это Error, наш заботливый @ExceptionHandler(Exception.class) просто постоит в сторонке и посмотрит, как сервис красиво горит. Ограничение на 10MB тут работает как подорожник (спасает только кэш логера). Нужно резать вложенность или хотя бы ловить Throwable, пока нас не положили файликом на пару килобайт.


## Используемые технологии

- Java 17+
- Spring Boot
- Hibernate
- PostgreSQL
- Maven
- Docker Compose


## Структура проекта

```text
xml-processing-platform/
└── xml2json-service/
```

## Запуск

```bash
cd xml2json-service
docker build -t xml2json-service .
docker run -p 8081:8081 xml2json-service
```
